# Databricks Data Engineering - Estudos

Este repositório contém materiais, exercícios e anotações relacionados ao estudo de Data Engineering utilizando Databricks.

O conteúdo é baseado em módulos práticos de aprendizado, incluindo tópicos como visão geral da plataforma Databricks, uso da ferramenta para engenharia de dados e scripts SQL utilizados durante os exercícios.

## Estrutura
- Módulos com conteúdos e exercícios
- Scripts SQL utilizados nas aulas
- Arquivos de apoio para os laboratórios

## Objetivo
Este repositório tem como objetivo servir como material de estudo e prática durante o aprendizado de Databricks e conceitos de Engenharia de Dados.
